# shadtheme

Generate more than +100M shadcn/ui themes, easily. Pick a color harmony, preview contrast-safe tokens, and ship a polished interface without touching a spreadsheet.

## Getting Started

Install dependencies and run the development server:

```bash
bun install
bun dev
```

Open [http://localhost:3000](http://localhost:3000) to see the result.

## Features

- **Theme Designer**: Interactive color harmony picker with live preview
- **Contrast-Safe Tokens**: Automatically generated tokens that meet accessibility standards
- **CSS Export**: Export ready-to-use CSS variables for your projects
- **Component Showcase**: Preview how your theme looks across all shadcn/ui components
